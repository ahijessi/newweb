## My Personal Website
This website was built using Vibe Coding via Google AI Studio.

Rather than using a generic template, the entire site was developed through multiple rounds of iterative refinement with AI to achieve a highly personalised result.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
